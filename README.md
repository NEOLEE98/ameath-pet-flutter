### English Readme | [中文说明](#中文说明)

A tiny Aemeath that lives on your screen. It roams, idles, and stays out of your way.

## What You Can Do
- Drag Aemeath around your screen
- Let it roam automatically with adjustable speed
- Switch Aemeath size
- Keep it always-on-top on desktop
- Control it from the tray menu (desktop)

## Platforms
- Windows
- macOS
- Android (floating overlay)

## How It Works
### Desktop
- Aemeath runs in a frameless always-on-top window.
- Use the tray icon for quick actions like Show/Hide, size, and speed.

### Android
- Aemeath runs as a floating overlay.
- You’ll be asked for “Display over other apps” permission.
- Use **Start Overlay** in the app to launch it.
- Use **Apply Settings** after changing size or speed.

## Settings
- Aemeath size (desktop)
- Desktop roam speed
- Mobile roam speed
- Android overlay scale
- Launch at startup (desktop)
- Show overlay debug (optional)

## Notes
- The Android overlay only updates after you tap **Apply Settings**.

## 中文说明

一个生活在你屏幕上的飞行雪绒，会闲逛、待机，不打扰日常使用。

### 你可以做什么
- 拖动小爱到任意位置
- 自动闲逛，速度可调
- 桌面端可调大小，安卓为悬浮窗缩放
- 桌面端常驻置顶
- 桌面端托盘菜单快速控制
- 安卓悬浮窗模式运行

### 支持平台
- Windows
- macOS
- Android（悬浮窗）

### 使用方式
#### 桌面端
- 以无边框置顶窗口运行
- 通过托盘菜单进行显示/隐藏、大小和速度调整

#### 安卓
- 以悬浮窗形式运行
- 首次使用需授予“显示在其他应用上层”
- 点击 **Start Overlay** 启动悬浮窗
- 修改大小或速度后点击 **Apply Settings** 生效

### 设置项
- 小爱大小（桌面端）
- 桌面闲逛速度
- 移动端闲逛速度
- 安卓悬浮窗缩放
- 开机自启动（桌面端）
- 显示调试信息（可选）

### 备注
- 安卓悬浮窗的设置需点击 **Apply Settings** 才会更新
